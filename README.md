# Magic Eight Ball

## Description
Magic Eight Ball is a React app based on the classic Magic Eight Ball toy.

How-To:
Type in your question. Click the circlular Ask button on the black magic eight ball circle. Wait while the magic ball thinks for a second â€¦ and spits out your answer!

Try it here: https://magic-eight-ball11.onrender.com/

## Getting Started

### Dependencies

* React / Vite / JavaScript
* HTML
* CSS

### Installing

1. Clone the repository:
  * git clone https://github.com/erica-b/magic-eight-ball.git
 

- Install dependencies:
  * npm install
 
### Executing program

1. **npm run dev** in your terminal
